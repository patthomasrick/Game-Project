Make sure you are using Internet Explorer. This DOES NOT work in Google Chrome or Firefox. Untested in Safari.

Not working?

To allow this applet to run, make sure:

  • This website is added to your Java security exceptions list. To do that, go to your Java settings (Configure Java in the Windows Start Menu), go to the Security tab, click Edit Site List... beside the Exception Site List, click Add, and then copy and paste or type https://patthomasrick.github.io/Game-Project/ into the box.
  • You are using Internet Explorer.
